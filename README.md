# Birder CLIP

Birder CLIP is a Birder extension for CLIP-style image-text models and related multimodal workflows.

This project is a work in progress. At the moment it contains only early package scaffolding.
Training code, inference workflows, model definitions and pretrained model integration are not available yet.

## Acknowledgements

Birder CLIP is built upon ideas, conventions and selected implementation details from [OpenCLIP](https://github.com/mlfoundations/open_clip), which serves as the primary reference implementation for CLIP-style image-text modeling in this project.
